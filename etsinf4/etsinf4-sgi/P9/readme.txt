CONTROLES:

1. Flecha izquierda/derecha: giro del vehículo
2. Flecha arriba / abajo: aumento / disminución de la velocidad
3. s : Activa / desactiva un modelo simple en alámbrico
4. l : Cambia entre modo diurno / nocturno
5. n : Cambia el estado de la niebla(on / off)
6. c : Cambia la visibilidad de elementos solidarios a la cámara(on / off):
		Velocimetro y cronometro
7. h : Cambia el modelo de vehículo
